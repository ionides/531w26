# Modeling and Analysis of Time Series Data (STATS 531)

## Chapter 18. A case study of ebola: Forecasting and diagnostics

We continue using material from the short course [Simulation-Based Inference for Epidemiological Dynamics](http://github.com/pypomp/tutorials/tree/master/sbied) (SBIED). STATS 531 Chapter 18 is Chapter 6 of SBIED. This analysis uses summary statistics, called _probes_, to diagnose model misspecification. A scientific goal is to investigate the advantages of using POMP models rather than deterministic mechanistic models. We show how to forecast using a POMP model.
