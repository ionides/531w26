
# Mini-quizzes

The data directory is a symbolic link to mt1/data, created by
```
ln -s ../mt1/data data
```

To set up the mini-quiz in Canvas

Quiz ->
  +Quiz ->
    Assignment name : Class x 
    Assignment Group : Participation
    Point : 2
    Due: 3:45PM
    Available from: 3:00PM
    Until: 3:45PM

  Build

Then set up the quiz, following class-x.txt
