Question text:
The question is posted at https://ionides.github.io/531w26/quiz/class19.pdf

Answers: 

(A). The code is okay, so there is some deeper problem. Perhaps we are working with an old version of Python (say, 3.8) and using a recent JAX version which is not fully backward-compatible, leading to a compilation error.

(B). Arguments to compiled functions must be JAX arrays. Replacing `10` by  
`jax.numpy.array([10])` will fix it. 

* (C). We cannot expect JAX to compile code for a variable value of random numbers. Set
`get_random_normal_jit = jax.jit(get_random_normal, static_argnames=['shape'])`

(D). We cannot reasonably expect JAX to compile for a variable `key`. We should set
`get_random_normal_jit = jax.jit(get_random_normal, static_argnames=['key'])`

(E). The problem will be solved if we use the @jit decorator to compile the code while keeping the same name as the original function.
