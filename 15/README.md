# Modeling and Analysis of Time Series Data (STATS 531)

## Chapter 15. Likelihood maximization for POMP models

We continue using material from the short course [Simulation-Based Inference for Epidemiological Dynamics](http://github.com/pypomp/tutorials/tree/master/sbied) (SBIED). STATS 531 Chapter 15 is Chapter 4 of SBIED. The main topic is likelihood maximization via an iterated particle filter. This enables a range of tools of likelihood-based inference to be applied---maximum likelihood estimation, likelihood ratio tests, profile likelihood confidence intervals, and AIC for model selection. Methods are demonstrated on a model for measles, but these techniques apply to the wide range of POMP models for which particle filtering is applicable.
